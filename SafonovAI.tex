			\begin{multline*}
				H^* = \varepsilon^2 \skobki{[}{.}{ a \tilde R_1^{3/2} \cos 3\tilde\Phi_1 + b \tilde R_1^{1/2}\tilde R_2 \cos \skobkiKr{\tilde\Phi_1 + 2\sigma \tilde\Phi_2} + } \qquadTri\\
				+\skobki{.}{]}{c_{20} R_1^2 + c_{11} R_1 R_2 + c_{02} R_2^2 } + O\skobkiKr{\varepsilon^3},
			\end{multline*}
			
			\item Производя замену $\tilde R_j = \cfrac{c_{11}^2}{b^2} R_j$ и, переходя к новому времени $\tilde \tau = \cfrac{b^2}{c_{11}} \tau$, получим гамильтонианы
